Claim:
If there is a plane $ax + by + cz = d$, then $\langle a, b, c \rangle$ is a vector normal to the plane.

Proof:
Any two points on the plane $(x_0, y_0, z_0)$ and $(x_1, y_1, z_1)$ satisfy $ax_0 + by_0 + cz_0 = d$ and $ax_1 + by_1 + cz_1 = d$.
Moreover the vector $\langle x_0 - x_1, y_0 - y_1 , z_0 - z_1 \rangle$ lies on the plane.
Thus
$\langle x_0 - x_1, y_0 - y_1 , z_0 - z_1 \rangle \cdot \langle a, b, c \rangle$
$=a(x_0 - x_1) + b(y_0 - y_1) + c (z_0 - z_1)$
$= ax_0 + by_0 + cz_0 - (ax_1 + by_1 + cz_1$)
$= d - d $
$= 0$
Since the dot product is 0, $\langle a, b, c \rangle$ is perpendicular to any vector on the plane.
This proof can be similarly extended to any nth dimensional plane for a nth dimensional vector.
