Claim:
Let us assume that the elements of an multidimensional array $V[x_{range}][y_{range}][z_{range}]$ are stored contiguously in memory.
Then $v[x][y][z]$ is stored at index $v + (x * y_{range} + y) * z_{range} + z$, where $v$ is a pointer to the first element in the array.

Proof:

In the diagram above we see that $x_{range} = 4$ , $y_{range} = 3$, and $z_{range} = 2$.
If we use a tree diagram, we reduce the problem to counting the number of nodes in the bottom layer.
The number of nodes to the left of $v[x][y][z]$ is calculated as $(x*3*2) + (y*2) + (z)$.
Take for example $v[0][1][1]$, the index will be $ v + (0*3*2) + (1*2) + (1) = v + 3$
More generally this is $ v + (x*y_{range}*z_{range}) + (y*z_{range}) + (z)$
Factoring this expression yields $v + (x * y_{range} + y) * z_{range} + z$.
This method of proof will work with any number of dimensions.
